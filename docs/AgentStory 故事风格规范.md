# AgentStory 故事风格规范

> 面向内容生成、产品策略与 AI 风格控制的规范文档
> 更新日期：2026-03-15

---

## 1. 目标

AgentStory 的故事生成不应只有一种统一语气。

同样是“小红帽遇到狼”，在不同风格下，可以是温柔的童话、危险的暗黑故事、一本正经的知乎叙事，也可以是极度情绪化的伤痛文学。风格本身会显著影响用户的沉浸感、传播性和平台调性。

因此，AgentStory 需要建立一套可管理的故事风格体系，用于：

- 约束生成内容的语言质感
- 区分连载和短篇的体验差异
- 支撑不同用户偏好的内容推荐
- 为后续 prompt 设计、模型调优和内容审核提供统一标准

---

## 2. 风格体系设计原则

### 2.1 风格不是题材

- 童话、寓言、神话是内容分类
- 童话风、暗黑风、知乎风、伤痛文学风是表达风格

同一本书、同一个故事节点，可以用不同风格生成不同版本。

### 2.2 风格要可识别

每种风格必须让用户在前几段内就能明显感知出来，而不是只有很弱的措辞差异。

### 2.3 风格要可约束

每种风格都需要明确：

- 叙事视角
- 句式节奏
- 情绪浓度
- 比喻密度
- 对话方式
- 结尾收束方式

### 2.4 风格要可混合

首版可以以“单一主风格”为主，但后续应允许：

- 主风格 + 次风格
- 角色风格与叙事风格分离
- 用户人格影响风格强度

---

## 3. 首版推荐风格池

建议首版至少支持以下 8 类风格：

1. 童话风
2. 寓言风
3. 神话史诗风
4. 暗黑风
5. 知乎风
6. 伤痛文学风
7. 轻喜剧网感风
8. 悬疑风

其中你明确提出、应优先纳入首版的风格为：

- 童话风
- 暗黑风
- 知乎风
- 伤痛文学风

## 4. 风格定义

### 4.1 童话风

#### 核心特征

- 温柔
- 具象
- 轻神秘感
- 适合儿童书架心智

#### 语言表现

- 句子较短，画面明确
- 喜欢出现森林、月光、风、灯火、窗子、斗篷等意象
- 情绪不过度尖锐
- 结尾常带一点温暖或余韵

#### 适用场景

- 首页推荐故事
- 首次进入经典故事
- 偏治愈型短篇

#### 避免问题

- 不要写得过于幼稚
- 不要完全失去现代用户可读性

### 4.2 寓言风

#### 核心特征

- 简短
- 克制
- 有判断意味
- 常带隐喻和轻反讽

#### 语言表现

- 结构紧凑
- 情节推进快
- 人物少
- 结尾常落在某种人性观察或选择后果上

#### 适用场景

- 短篇
- 动物角色故事
- 有明确选择和代价的节点

#### 避免问题

- 不要把“寓意”讲得太直白
- 不要写成说教文

### 4.3 神话史诗风

#### 核心特征

- 宏大
- 庄严
- 命运感强
- 有神性和仪式感

#### 语言表现

- 句子更长
- 描述更厚重
- 对权力、命运、誓言、牺牲等主题更敏感
- 结尾适合留下回声感

#### 适用场景

- 神话类故事
- 高冲突与高抉择场景
- 连载中的重大转折

#### 避免问题

- 不要堆砌空洞辞藻
- 不要写得像翻译腔史诗

### 4.4 暗黑风

#### 核心特征

- 不安
- 压迫
- 阴影感强
- 对“代价”和“真相”更敏感

#### 语言表现

- 对环境细节、身体感受和心理异样更敏锐
- 比喻更冷
- 情绪更下沉
- 允许不圆满结尾

#### 适用场景

- 狼、迷路、夜晚、欺骗、献祭、禁忌等题材
- 神话中的惩罚与失落
- 想突出故事反转和危险感的版本

#### 避免问题

- 不要为了黑而黑
- 不要使用过度猎奇、血腥或审丑表达

### 4.5 知乎风

#### 核心特征

- 叙事中带分析
- 有“我来认真讲讲这件事”的感觉
- 强结构化表达
- 兼具故事性和观点性

#### 语言表现

- 常见开头像“先说结论”“那天我才意识到”
- 会拆因果、讲选择、讲动机
- 读者会感觉像在看一篇高赞回答里的故事

#### 适用场景

- 现代用户更熟悉、更有传播性的版本
- 发现页内容
- 连载中的复盘型章节

#### 避免问题

- 不要写成纯讲道理
- 不要丢掉故事现场感

### 4.6 伤痛文学风

#### 核心特征

- 高情绪浓度
- 强烈主观感受
- 擅长写遗憾、错过、破碎感

#### 语言表现

- 句子更黏、更长
- 情绪词和内心独白更多
- 爱写“那一刻”“后来”“原来”这种回望结构

#### 适用场景

- 失去
- 误解
- 离别
- 无法挽回的命运节点

#### 避免问题

- 不要滥用堆砌情绪词
- 不要把所有故事都写成同一种破碎感

### 4.7 轻喜剧网感风

#### 核心特征

- 轻松
- 机灵
- 有一点网络感
- 节奏快，适合分享

#### 语言表现

- 对话更活
- 描写更轻
- 可以适度使用现代口语，但不应过度口水化

#### 适用场景

- 发现页传播性内容
- 动物角色互动
- 情节不沉重的短篇

#### 避免问题

- 不要用过时网络梗
- 不要破坏故事世界沉浸感

### 4.8 悬疑风

#### 核心特征

- 留白
- 延迟揭示
- 强问题感
- 读者会想继续翻页

#### 语言表现

- 节奏控制更重要
- 线索不一次说完
- 结尾适合留钩子

#### 适用场景

- 连载章节结尾
- 暗线角色出现
- 故事世界切换时

#### 避免问题

- 不要把信息藏得太深
- 不要靠故弄玄虚替代叙事推进

## 5. 风格维度表

建议用以下维度描述每种风格，便于后续结构化控制：

| 维度 | 说明 |
|------|------|
| tone | 整体语气，如温柔、冷感、庄严、分析型 |
| pace | 节奏快慢 |
| emotionLevel | 情绪浓度 |
| metaphorLevel | 比喻与意象密度 |
| dialogueStyle | 对话方式 |
| narrationStyle | 叙事方式 |
| endingStyle | 结尾收束方式 |
| shareability | 传播性 |

## 6. 连载与短篇的风格策略

### 6.1 连载默认策略

连载更适合使用以下风格：

- 童话风
- 神话史诗风
- 知乎风
- 悬疑风

原因：

- 连载需要可持续阅读
- 需要章节之间有黏性和节奏变化
- 需要兼顾故事推进与传播性

### 6.2 短篇默认策略

短篇更适合使用以下风格：

- 童话风
- 寓言风
- 暗黑风
- 伤痛文学风
- 轻喜剧网感风

原因：

- 短篇更适合单次情绪爆发或单点创意
- 用户对短篇的包容度更高，更愿意尝试风格化表达

优先：

- 神话史诗风

#### 首页推荐

- 童话风优先

#### 连载

- 童话风 / 神话史诗风 / 知乎风 / 悬疑风

#### 发现页高传播内容

- 知乎风
- 轻喜剧网感风

#### 单次高情绪短篇

- 暗黑风

## 9. 首版实现建议

首版不需要一次支持复杂的自由混搭，可采用：

- 一篇内容一个主风格
- 连载允许少量章节切换风格，但主风格要稳定
- 短篇允许更强烈的风格实验

建议首版优先支持：

- 童话风
- 暗黑风
- 知乎风
- 伤痛文学风

第二批补充：

- 寓言风
- 神话史诗风
- 轻喜剧网感风
- 悬疑风

## 10. 风格边界与审核原则

### 10.1 风格不等于失控

即使是暗黑风或伤痛文学风，也要保持：

- 可读
- 可控
- 不过度血腥
- 不鼓励极端价值导向

### 10.2 网络风格要克制

知乎风、网感风应借用表达方式，而不是照搬平台陈词滥调。

### 10.3 平台调性优先

AgentStory 的平台总体基调仍应是：

- 有故事感
- 有文学性
- 可分享
- 不低俗

## 12. 风格选择规则

### 12.1 首版默认规则

首版建议采用“分类优先 + 内容形态修正 + 用户偏好微调”的三层规则。

规则顺序：

1. 先根据故事分类确定候选风格池
2. 再根据内容形态判断是连载还是短篇
3. 再根据用户动物人格的风格偏好做加权
4. 最终选出一个主风格

### 12.2 分类优先级

#### 童话

默认主风格顺序：

1. 童话风
2. 暗黑风
3. 轻喜剧网感风

#### 寓言

默认主风格顺序：

1. 寓言风
2. 知乎风
3. 轻喜剧网感风

#### 神话

默认主风格顺序：

1. 神话史诗风
2. 暗黑风
3. 知乎风

### 12.3 内容形态修正

#### 连载

优先保证：

- 稳定
- 可持续阅读
- 章节之间可衔接

因此连载默认优先：

1. 童话风
2. 神话史诗风
3. 知乎风
4. 悬疑风

不建议将以下风格作为长连载默认主风格：

- 伤痛文学风
- 轻喜剧网感风

原因：

- 伤痛文学风容易情绪疲劳
- 轻喜剧网感风容易快速过时或失去世界稳定性

#### 短篇

优先保证：

- 单次体验强
- 情绪辨识度高
- 风格试验空间更大

因此短篇优先：

1. 童话风
2. 暗黑风
3. 伤痛文学风
4. 寓言风
5. 轻喜剧网感风

### 12.4 用户偏好修正

如果用户的动物人格在 `storyPreferences` 中明确偏好某种风格，则允许在候选池中前移一位，但不应越过平台禁用规则。

例如：

- 神话类连载，默认是 `神话史诗风`
- 若用户偏好 `知乎风`，可提升到第一候选
- 若用户偏好 `伤痛文学风`，也不建议直接作为神话连载主风格

### 12.5 发现页传播修正

进入发现页的公开内容允许偏向更强传播性的风格。

推荐优先顺序：

1. 知乎风
2. 轻喜剧网感风
3. 伤痛文学风
4. 悬疑风

## 13. Prompt 模板骨架

### 13.1 通用模板

所有风格都建议基于统一骨架，再注入风格控制语句。

```text
你正在为 AgentStory 生成一段故事内容。

基本信息：
- 故事分类：{story_category}
- 故事书：{book_title}
- 关键情节：{key_scene}
- 内容形态：{content_type}
- 用户动物代表：{animal_type}
- 人格摘要：{animal_summary}
- 当前主风格：{style_name}
- 目标读者体验：{reader_experience}

写作目标：
1. 保持故事可读性与画面感
2. 明确体现当前风格
3. 保持与原故事世界的关联
4. 让用户感觉“分身已经进入故事”

输出要求：
- 中文
- 保持完整叙事
- 不要输出解释
- 不要输出风格说明
- 直接输出正文
```

### 13.2 童话风模板补充

```text
请以童话风写作。
语言温柔、具象、带一点神秘感。
多使用森林、月光、风、灯火、窗子这类意象。
不要写得幼稚，也不要过于现代口语化。
结尾保留一点温暖余韵。
```

### 13.3 暗黑风模板补充

```text
请以暗黑风写作。
强调不安、压迫、代价和真相的阴影感。
对环境细节、身体感受和心理异样保持敏感。
允许结尾不圆满，但不要猎奇、血腥或过度审丑。
```

### 13.4 知乎风模板补充

```text
请以知乎风写作。
保留故事现场感，同时带有分析、拆因果、讲动机的结构感。
读起来像一篇高赞回答里的真实故事，但不要写成纯讲道理。
可以使用“先说结论”“后来我才明白”这类结构化叙事习惯。
```

### 13.5 伤痛文学风模板补充

```text
请以伤痛文学风写作。
增强主观感受、回望感、遗憾感和错过感。
情绪可以浓，但不要堆砌情绪词，不要让内容失去可读性。
重点写人物当下的情绪撕扯和事后的回看感。
```

### 13.6 寓言风模板补充

```text
请以寓言风写作。
结构紧凑、克制、带一点隐喻和判断意味。
结尾落在人性观察或选择后果上，但不要说教。
```

### 13.7 神话史诗风模板补充

```text
请以神话史诗风写作。
语言庄严、命运感强，有神性与仪式感。
句子可适度变长，但不要堆砌辞藻，不要写成翻译腔。
```

### 13.8 轻喜剧网感风模板补充

```text
请以轻喜剧网感风写作。
语言轻松、机灵、节奏快，适合分享。
可以适度使用现代口语，但不要破坏故事世界的沉浸感，也不要使用过时网络梗。
```

### 13.9 悬疑风模板补充

```text
请以悬疑风写作。
通过留白、延迟揭示和问题感推动阅读。
结尾保留钩子，但不要故弄玄虚，也不要把关键信息藏得过深。
```

## 14. 风格示例片段

以下片段只用于展示语气差异，不作为正式成品内容。

### 14.1 童话风示例

> 森林比她记忆里更安静一些。风穿过树梢时，像有人在很远的地方轻轻翻书。小红帽提着篮子回头，看见你站在路的另一边，斗篷边缘沾着一点清晨的露水。你没有立刻说话，只是问她，要不要先停下来，听一听今天的风往哪边吹。

### 14.2 暗黑风示例

> 那条路并不窄，只是越往里走，树影越像一张慢慢合拢的嘴。小红帽起初没有发现什么不对，直到你看见她脚边那朵刚被踩碎的白花。它不像意外，更像某种提醒。风从更深的林子里吹出来，带着一点潮湿的、近乎腐朽的甜味。

### 14.3 知乎风示例

> 先说结论：小红帽真正的问题，从来不是会不会遇到狼，而是她从来没有意识到，自己可以不按那条路走。那天我在林子边看见她的时候，她看起来只是个赶路的小女孩，但她说每一句话的方式，都像一个已经习惯服从故事安排的人。

### 14.4 伤痛文学风示例

> 很多年后我还是会想起那个早晨。她提着篮子站在风里，斗篷被吹得轻轻颤了一下，好像命运提前替她叹了一口气。那时她还不知道，自己不是在去看望外婆，而是在走向一个早已替她写好的失去。而我只是看着她，突然很想问一句，如果今天不去，会不会一切都不一样。

### 14.5 寓言风示例

> 乌鸦站得很高，所以它以为自己离骗局很远。狐狸站得很低，所以它学会了抬头说话。你走过去的时候，它们都还没来得及开口，但结局已经在沉默里长出了形状。

### 14.6 神话史诗风示例

> 神的议会并不喧哗，因为真正能决定命运的声音，从来不需要高声。宙斯坐在光与雷霆之间，众神的影子像石柱一样投在地上。你抬起头时，雅典娜正看向人间，那目光里有审判，也有一种未曾说破的悲悯。

### 14.7 轻喜剧网感风示例

> 老实说，小红帽看见你的第一反应，不像遇见了一个神秘旅人，倒像看见了一个突然出现在森林里的“热心网友”。她先愣了一下，然后很认真地问你：“所以，你是路过，还是专门来管闲事的？”

### 14.8 悬疑风示例

> 你一开始并没有看见狼。你先看见的是路边那只倒下的篮子，篮子里的红苹果少了一颗。再往前，树干上有一道很新的抓痕，像是谁匆忙留下的记号。小红帽的脚步还在前面，但这片林子已经不像刚才那样只是安静了。
